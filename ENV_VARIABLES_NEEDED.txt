========================================
  ADD THESE TO YOUR .env.local FILE
========================================

Copy and paste these lines into your .env.local file:

# Cloudinary Configuration (NEW - Required for image uploads)
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=your_cloud_name_here
NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET=whitelabel-uploads

========================================
  HOW TO GET THESE VALUES
========================================

1. NEXT_PUBLIC_MAPBOX_ACCESS_TOKEN:
   - Go to https://account.mapbox.com/
   - Sign up or log in to your Mapbox account
   - Go to your Account page (https://account.mapbox.com/access-tokens/)
   - Copy your Default public token or create a new one
   - Make sure it has "Geocoding API" scope enabled
   - Example: "pk.eyJ1IjoieW91cnVzZXJuYW1lIiwiYSI6ImN..."

2. NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME:
   - Go to https://cloudinary.com/console
   - Login to your account
   - Copy the "Cloud name" from your dashboard
   - Example: "dxyz123abc"

2. NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET:
   - In Cloudinary dashboard, go to Settings → Upload
   - Scroll to "Upload presets"
   - Click "Add upload preset"
   - Name it: "whitelabel-uploads"
   - Set Signing Mode to: "Unsigned" ⚠️ IMPORTANT
   - Click Save
   - Use this exact name: "whitelabel-uploads"

========================================
  FACEBOOK MESSENGER BOT SETUP
========================================

1. FACEBOOK_PAGE_ACCESS_TOKEN:
   - Go to https://developers.facebook.com/
   - Create a new app or use existing
   - Add "Messenger" product to your app
   - Go to Messenger → Settings → Access Tokens
   - Select your Facebook Page
   - Click "Generate Token"
   - Copy the Page Access Token
   - Example: "EAAxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"

2. FACEBOOK_VERIFY_TOKEN:
   - This is a custom token you create (any string)
   - Used for webhook verification
   - Should be a random, secure string
   - Example: "my_secure_verify_token_12345"
   - Keep it secret!

3. FACEBOOK_APP_SECRET:
   - Go to your Facebook App → Settings → Basic
   - Find "App Secret"
   - Click "Show" and copy it
   - Used for webhook signature verification
   - Example: "abc123def456ghi789..."

4. Set up Webhook:
   - In Facebook App → Messenger → Settings → Webhooks
   - Callback URL: https://your-domain.com/api/messenger/webhook
   - Verify Token: Use the same value as FACEBOOK_VERIFY_TOKEN
   - Subscribe to: messages, messaging_postbacks
   - Save

5. Subscribe Page to Webhook:
   - In Webhooks section, click your webhook
   - Click "Subscribe" next to your Page
   - Now your bot will receive messages!

========================================
  GOOGLE MAPS SETUP
========================================

1. Go to https://console.cloud.google.com/
2. Create a new project or select existing
3. Enable "Places API"
4. Go to Credentials → Create API Key
5. Restrict the key to "Places API" for security
6. Copy your API key
7. Add to .env.local

Note: Google Places has better coverage of commercial establishments,
shopping malls, and landmarks in Philippines. Use as fallback when
Mapbox doesn't find specific places.
