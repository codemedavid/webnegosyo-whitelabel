## ğŸš¨ Common Issues & Solutions

### Issue 1: Colors Not Updating

**Symptom**: Changed colors in admin don't reflect on frontend

**Causes**:
1. Browser cache
2. SSR caching
3. React Query stale cache

**Solutions**:
```typescript
// 1. Hard refresh browser (Cmd+Shift+R)

// 2. Revalidate in server action
revalidatePath(`/${tenant.slug}/menu`)

// 3. Invalidate React Query
queryClient.invalidateQueries(['tenant', slug])
```

### Issue 2: Tenant Admin Can't Edit Settings

**Symptom**: "You don't have permission" error

**Causes**:
1. Not logged in as admin
2. Wrong tenant assignment
3. RLS policy blocking

**Solutions**:
```sql
-- Check user role
SELECT * FROM app_users WHERE user_id = auth.uid();

-- Verify tenant assignment
SELECT tenant_id FROM app_users WHERE user_id = auth.uid();

-- Check if tenant matches
SELECT id FROM tenants WHERE slug = '[tenant-slug]';
```

### Issue 3: Lalamove Not Working

**Symptom**: Delivery option not showing

**Checklist**:
- [ ] `lalamove_enabled = true`
- [ ] API keys configured
- [ ] Restaurant address filled
- [ ] Order type "delivery" exists
- [ ] Order type is enabled

**Debug**:
```typescript
// Check tenant config
console.log(tenant.lalamove_enabled)
console.log(tenant.restaurant_address)
console.log(tenant.lalamove_api_key ? 'Key set' : 'No key')
```
