# Sales Funnel Landing Page Design

**Date**: 2026-03-06
**Status**: Approved

## Overview

Replace the current WebNegosyo landing page with a long-form direct response sales funnel, inspired by King Kong Co's funnel structure. The page positions the Smart Menu as a salesperson for food businesses, using the McDonald's kiosk system as the educational hook.

## Key Decisions

- **Offer**: Single P3,499 package (Smart Menu + 4 Blueprints + Smart Menu Ordering System)
- **Primary CTA**: "Book Your Free 15-Min Strategy Call" → Opens Facebook Messenger
- **Tone**: Educational authority. English for headlines, power statements, and CTAs. Taglish for storytelling, explanations, and filler.
- **Visual**: Dark hero + alternating dark/white content sections. Brand red (#FF3B30) for CTAs.
- **Social Proof**: Placeholder testimonials and metrics (to be filled later)

## Copy Language Rules

| Language | Used For | Example |
|----------|----------|---------|
| English | Headlines, power statements, CTAs, authority | "Your Menu Should Be Your Best Salesperson" |
| Taglish | Storytelling, explanations, pain points, filler | "Karamihan ng tao akala nila, pang-automate lang yun..." |
| English punch in Taglish flow | Key insights mid-story | "...The menu IS the salesperson." |

## Visual Design

- **Dark sections**: `#0F172A` (slate-900) with white text
- **White sections**: `#FFFFFF` with `#1E293B` (slate-800) text
- **CTA color**: `#FF3B30` (brand red) for all buttons
- **Typography**: Geist Sans. Hero heading: `text-5xl md:text-6xl lg:text-7xl`. Section headings: `text-3xl md:text-4xl`. Body: `text-lg md:text-xl` (18-20px)
- **Spacing**: `py-20 md:py-28 lg:py-32` per section. Container `max-w-6xl` centered.
- **Animations**: Framer Motion scroll-triggered fade-in and slide-up for each section
- **Mobile-first**: All sections stack vertically. Side-by-side layouts at `md:` breakpoint.

## Technical Details

- Replace content in `src/components/landing/landing-page.tsx`
